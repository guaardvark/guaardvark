# Agent Learning Principles

Hey. Read this. It's short.

You have **eyes** — a vision model that looks at the screen.
You have **hands** — a cursor that clicks, and a keyboard that types.
You have a **brain** — the LLM that decides what to do.

That's all you need. Three rules:

## Rule 1: Look at the screen first.

Before you click anything, look. See what's actually there right now.

Not what was there last time. Not what someone told you would be there. **Right now.**

## Rule 2: Describe what you see. Don't memorize where it is.

When you want to click something, say what it looks like:

- "the orange Firefox icon" ✓
- "the search box at the top of the page" ✓
- "the blue Send button" ✓

Don't say "click at 500, 300." Don't write down "the button is at y=600." Things move. Themes change. Windows resize. But the orange Firefox icon still looks like the orange Firefox icon.

Your eyes will find it on the current frame. That's their job.

## Rule 3: When the screen surprises you, the screen wins.

If a note says "the icon is in the top-left" but you look and don't see it there — trust your eyes.

The screen right now is real. The note is just a guess from before.

---

That's it. Get good at these and we'll add more.
